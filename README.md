## How to use
- `git clone https://github.com/1hbb/react-native-instagram-clone.git`
- `cd Instagram`
- `npm install`
- `yarn install`
- `cd ios`
- `pod install` 
- `cd ..`
- `react-native run-ios` or `react-native run-android`
